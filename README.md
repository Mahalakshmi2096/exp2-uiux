## EXP 2 - UIUX
EX-2-Design a Basic Mobile and Web Layout for an Existing Case Study

## AIM:
To design and prototype a BeautyLuxe Beauty Product Shopping Application consisting of a Login Page, Web Home Page, and Mobile Hom Page using Canva, while applying fundamental UI/UX design principles and Gestalt principles to create a visually appealing, user-friendly, and responsive interface.

## ALGORITHM:

### Step 1: Define Project Requirements
Identify the purpose of the application (Beauty Product Shopping Platform).
Determine the target users (beauty and skincare customers).
List the required screens:
Login Page, Web Home Page, Mobile Home Page

### Step 2: Create a Canva Design Project
Open Canva.
Select appropriate dimensions:
Web Layout: 1366 × 768 px
Mobile Layout: 390 × 844 px
Create separate pages for each screen.

### Step 3: Design the Login Page
Select a cream/beige background.
Add the BeautyLuxe logo and tagline.
Insert beauty product illustrations or beauty-themed graphics.
Add: Username text field, Password text field, Login button, Sign-Up button
Apply rounded corners and soft shadows.
Align all elements centrally.

### Step 4: Design the Web Home Page
Create a navigation bar.
Add a hero banner featuring beauty products.
Insert a call-to-action button (Shop Now).
Create category cards
Add product cards
Design a simple footer section.

### Step 5: Design the Mobile Home Page
Create a mobile-sized canvas.
Add a header 
Add category icons in a horizontal arrangement.
Create product cards for featured products.
Add a bottom navigation bar:

### Step 6: Apply UI/UX Principles
Use consistent colors throughout all screens.
Maintain proper spacing and alignment.
Use readable typography.
Ensure visual hierarchy by emphasizing important elements.
Keep navigation simple and intuitive.

### Step 7: Apply Gestalt Principles
Use Proximity to group related elements.
Use Similarity for buttons, cards, and icons.
Use Figure-Ground to highlight interactive components.
Use Common Region by placing content inside cards.
Use Continuity through proper alignment and layout flow.

### Step 8: Review and Refine
Check consistency across all screens.
Verify alignment and spacing.
Ensure buttons and navigation elements are clearly visible.
Make necessary design improvements.

### Step 9: Export the Design
Click Share in Canva.
Select Download.
Export the designs as PNG or PDF.
Save the final design files.
